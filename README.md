# AI Meeting Notes Summarizer

An AI-powered application that converts meeting transcripts into structured summaries, key discussion points, action items, and important decisions using Google's Gemini LLM.

## Features

- Upload meeting transcripts (.txt)
- Generate executive summaries
- Extract key discussion points
- Identify action items
- Highlight important decisions
- Streamlit-based user interface

## Tech Stack

- Python
- Streamlit
- Google Gemini API
- python-dotenv

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
